# Derivatives

Unknown

\\( 
    f'(x) = ?
\\)

Given

\\( 
    f(x) = \frac{x^2}{2} + C  
\\)

Solution

\\( 
    f'(x) = 
    \frac{dy}{dx} ( f(x) ) = 
    \frac{dy}{dx} ( \frac{x^2}{2} + C ) =
    \frac{nx^{n-1}}{2} + 0 = 
    \frac{2x^{1}}{2} + 0 = 
    x 
\\)
